# Exercícios de Vetores em JavaScript

## Descrição

Este repositório contém a resolução de exercícios desenvolvidos em **JavaScript**, com foco na utilização de **vetores** para armazenamento e manipulação de dados.

Os programas foram implementados sem utilizar funções próprias de arrays, priorizando a construção dos algoritmos de forma manual para reforçar os conceitos de lógica de programação.

## Objetivos da Atividade

* Desenvolver programas em JavaScript (`.js`);
* Aplicar estruturas de vetores para resolução de problemas;
* Testar e validar os algoritmos desenvolvidos;
* Utilizar Git para versionamento local;
* Publicar os arquivos em um repositório remoto no GitHub.

## Exercícios Desenvolvidos

### 1. Maior idade entre 4 pessoas

O programa:

* Lê a idade de 4 pessoas;
* Armazena as idades em um vetor;
* Exibe:

  * A maior idade informada;
  * A posição em que a maior idade foi digitada.

### 2. Cadastro de funcionários

O programa:

* Lê os dados de 5 funcionários:

  * Nome;
  * Sexo;
  * Salário;
* Armazena as informações em três vetores distintos;
* Exibe apenas as funcionárias do sexo feminino que recebem mais de R$ 5.000,00.

### 3. Cadastro de alturas

O programa:

* Permite cadastrar nomes e alturas de várias pessoas;
* Continua a leitura até que o usuário escolha encerrar;
* Ao final, apresenta:

  * A maior altura e o nome da pessoa correspondente;
  * A menor altura e o nome da pessoa correspondente.
